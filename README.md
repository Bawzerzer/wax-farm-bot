## Installation
Install nodejs and npm
> https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

Then run below script
```sh
$ git clone https://github.com/srisaard/wax-farm-bot.git
$ cd wax-farm-bot
$ npm install express request
$ node .
> server on6969
```
